Сайт для проекта "Создание RPG игры"

Описание модулей

  1. Управление главным персонажем

    • Модуль обработки пользовательского ввода для управления
    • Модуль логики поведения оружия

  2. Взаимодействие с миром

    • Модуль рендеринга и вывода сообщений
    • Модуль физического взаимодействия с объектами мира

  3. Онлайн игра

    • Модуль вторжения в другие миры, и "Игрок против игрока"

  4. Сюжетное повествование

    • Модуль проигрыша заставок и сюжетных видео
    • Модуль взаимодействия с неигровыми персонажами и их историей
